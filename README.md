# CafeApp

A simple Java console application for a café order system.

## Features

* Show café menu
* Customer enters name
* Choose item and quantity
* Loyalty member discount
* VAT calculation
* Print receipt

## Technologies

* Java
* IntelliJ IDEA
* Git & GitHub

## Menu

1. Espresso – 25 SEK
2. Cappuccino – 35 SEK
3. Latte – 40 SEK
4. Croissant – 30 SEK
5. Sandwich – 55 SEK

## How to Run

1. Open the project in IntelliJ IDEA
2. Run `CafeApp.java`
3. Follow the instructions in the console
